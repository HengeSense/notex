NoTex

A web based re-structured text editor

Download this project as a .zip file Download this project as a tar.gz file

The NoTex editor helps you to write reports using the lightweight markup language re-structured text aka RST. Once you've written your report, it can easily be translated to various formats like HTML, LaTex and PDF. To launch the application, use the following link:

Launch NoTex will start the application. This software is still beta!

Re-Structured Text

Purpose

RST is commonly used for technical documentation, e.g. in documentation of software libraries. However, re-structured text is also very much suitable for other kind of texts.

Tutorial

There are many good primers, tutorials and references about RST, e.g. RST Primer is quite a thorough introduction and Quick Re-Structured Text is a good quick reference.

Miscellanea

NoTex' front-end has entirely been written with the ExtJS JavaScript and the back-end with the http://www.djangoproject.com Python library. The back-end applies the Python document generator Sphinx to translate the RST input files to the various target formats.
